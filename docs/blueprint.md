# **App Name**: GradeAI

## Core Features:

- Dynamic Course Input: Enables users to dynamically add and remove course containers with fields for course name, credit hours, and marks, enhancing input flexibility.
- GPA Calculation: Calculates the GPA based on credit hours and marks entered for each course, updating in real-time as new courses are added or data is modified.
- Real-time GPA Visualization: Presents the calculated GPA visually using a doughnut chart that updates in real-time, offering immediate insights into academic performance.
- Grade Prediction Tool: Uses AI to predict potential grades based on the current input, allowing users to see the impact of different grades on their overall GPA. It's designed to assist in goal-setting and academic planning; the AI is acting as a tool for exploring hypothetical scenarios.
- Theming: The app uses multiple color themes to allow for user customization.

## Style Guidelines:

- Primary color: Deep blue (#2E4057), evokes trust and professionalism, fitting for an educational tool.
- Background color: Light gray (#EAEAEA), provides a clean and unobtrusive backdrop that enhances readability.
- Accent color: Teal (#20A39E), offers a fresh and energetic contrast, highlighting interactive elements and key data points.
- Body and headline font: 'Inter', sans-serif, to offer a modern and neutral look which enhances readability and a clean aesthetic.
- Use clear and simple icons to represent course categories, actions (add, delete), and GPA metrics.
- Adopt a clean, minimalist layout inspired by ChatGPT, ensuring ease of navigation and a user-friendly experience. Prioritize essential features and display information in an intuitive manner.
- Use subtle animations for feedback on user actions like adding or deleting courses and updating the GPA chart. Keep animations brief and purposeful to enhance engagement without being distracting.
